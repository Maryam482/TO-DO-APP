# TO-DO-APP

A simple todo list to manage tasks built using react and redux toolkit

### [Live Demo](https://todo-app-project4.netlify.app/)

## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Maryam482/TO-DO-APP.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
### Run
 ```sh
   npm start
   ```
